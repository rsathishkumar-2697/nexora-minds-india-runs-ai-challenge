# Intelligent Candidate Discovery & Ranking

## Team Name
Nexora Minds

## Challenge
INDIA.RUNS Data & AI Challenge

## Problem Statement
Build an AI-powered candidate ranking system that identifies the most suitable candidates for a job description using semantic understanding, career history, skills, and behavioral signals.

## Approach
- Candidate Profile Analysis
- Semantic Matching
- Hybrid Scoring
- Behavioral Signal Ranking
- Final Candidate Ranking

## Technologies
- Python
- Pandas
- Sentence Transformers
- FAISS
- Scikit-Learn

## Output
Top 100 ranked candidates with score and reasoning.
